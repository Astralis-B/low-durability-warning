# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-22
### Initial Release
- **Added**: Three customizable warning levels (First, Second, Critical)
- **Added**: Support for tools, weapons, and all armor types
- **Added**: Separate toggle for Elytra durability alerts
- **Added**: Colored visual alerts with custom text
- **Added**: Audible bell alerts with adjustable pitch and volume
- **Added**: Enchanted-only item filter option
- **Added**: Alert cooldown system to prevent spam
- **Added**: ModMenu integration for in-game configuration
- **Added**: Scheduled sound system for delayed alerts
- **Technical**: Client-side only implementation
- **Technical**: Built for Minecraft 1.21.11 with Fabric
